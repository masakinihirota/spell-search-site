# このアプリの概要

## 対象

スペルトナエル

## 説明

歌ビルドの歌を探しやすくするWebアプリです。

このアプリでは、以下の機能を提供します：
- 歌の検索機能
- 歌の詳細情報の表示
- お気に入りリストの作成と管理
